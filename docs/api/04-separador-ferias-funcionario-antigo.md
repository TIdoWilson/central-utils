# 04 - Separador Férias por Funcionário

## 🎯 Objetivo

Organizar dados de férias estruturados por funcionário.

---

## 📥 Entrada
- Excel (.xlsx) ou CSV com dados de férias
- Colunas: Matrícula, Nome, Empresa, Período, Dias

---

## 📤 Saída
- Estrutura organizada por funcionário
- Relatório consolidado (JSON/Excel)

---

## 🌐 Endpoint

```
POST /api/separador-ferias-funcionario/processar
```
