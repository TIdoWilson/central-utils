Conversor de PDF → PDF/A

Envie um arquivo PDF (ou outro formato suportado) e receba a versão PDF/A.

Enviar arquivo

O arquivo será convertido para PDF/A e ficará disponível para download.

Observações

  • Arquivos grandes podem levar alguns minutos.
  • Para outros formatos, o servidor converte primeiro para PDF.
  • O resultado é disponibilizado para download após a conversão.
